# Pudim News

Esse é um projeto sem fins lucrativos desenvolvido para a disciplina de Programação Web da Universidade Católica de Brasília. O objetivo deste projeto é reaprender a utilizar o HTML e o CSS. <br>

Utilizar "Pudim" como o tema veio da piada com o site pudim.com.br, um site que serve somente para segurar o domíno "pudim" e colocá-lo à venda. <br>

Projeto feito pelo aluno Luís Fernando Ventura.
